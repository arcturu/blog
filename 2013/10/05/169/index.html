vim&C++ 忘備録その１

Oct 5, 2013

随分と長い間プログラミングから離れていましたが、最近になってようやく勉強し直し始めました。この際なのでvimも使い始めてみました。そのなかで出会った問題の解決策を主に自分のためにまとめました。

1. C++コンパイラがない（OSX Lion）

どうやら最近のmacには標準でインストールされていないようなので、gccをXcode経由でインストールする。

Macにgccをインストールする

2. vim でクリップボード経由のコピペができない

mac 標準の vim は -clipboard でビルドされているせいで set clipboard = unnamed, autoselect などをしてもクリップボード経由のコピーアンドペーストができない。macvim をインストールするのも面倒なので何とかならないか調べたところ

Mac標準のvimでクリップボードが簡単に使えるように設定した

こんな素敵な記事を見つけたので一件落着。

3. 【未解決】solarized でビジュアルモードの色が変

ターミナルの標準色ではなんだかイメージしていた vim と違うのでカラースキーム solarized を適用してみたのだが、ビジュアルモードで選択部分の色がおかしい。シンタックスハイライトが消え、文字色と文字の背景色が共に似たような灰色になってしまうので非常に視認性が悪いのだ。色分けが消えるくらいならともかく文字が見えないのは不便なので何とかならないのか調べてみたところ

vim solarized in visual mode loses syntax coloring in selection

同じ症状にあっていると思われる人を発見。ただこの answer を試してみてもあまりしっくり来なかったので .vim/colors/solarized.vim の

exe "hi! Visual" .s:fmt_none .s:fg_base01 .s:bg_base03 .s:fmt_revbb

を

exe "hi! Visual" .s:fmt_none .s:fg_base02 .s:bg_base00 .s:fmt_revbb

に変え、とりあえず文字が見える状態にした。

これでひと通り開発環境が整ったはず。vim を使うことになるんだったら6,500円も出してcoda2を買ったのが馬鹿みたい、というか馬鹿だなあ……。
